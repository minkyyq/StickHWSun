Ядро Солнца (Якорь возрождения) — это блок, с которого через определённые промежутки времени (20–60 секунд) выпадают ресурсы.
-  Здесь можно получить множество предметов.
-  Помимо этого, в радиусе 12 блоков от Ядра Солнца вы будете получать монетки или опыт — они меняются местами в среднем каждые 25 минут.
